Giddy Web
======
**Giddy** not live yet.

## Setup
1. Install **ruby**, **bundler**, **rails** and **postgresql**. ([check tutorial](https://gorails.com/setup))
2. Install dependencies.

        bundle install

3. Setup database.

        rails db:setup

4. Start hacking.

        rails s
